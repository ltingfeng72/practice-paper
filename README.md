# 练习卷仓库

本仓库包含数据结构与算法课程的练习卷及其答案。

## 文件说明

- **练习卷1.pdf** - 数据结构与算法练习卷（共5大题，总分100分）
- **练习卷1-答案.md** - 练习卷1的完整答案

## 练习卷1内容概览

### 一、应用题（20分）
1. 单链表操作（节点交换、尾插法创建链表）
2. 顺序栈定义

### 二、应用题（20分）
1. 二维数组存储地址计算、上三角矩阵压缩存储
2. 广义表操作（表头、表尾、深度计算）

### 三、分析题（20分）
1. 二叉树遍历（先序、中序、后序）及森林还原
2. 顺序栈基本操作（判空、判满、入栈、出栈）

### 四、算法题（20分）
1. 图的回路检测算法（基于邻接表的DFS）
2. 平衡二叉树（AVL树）构造及查找长度计算

### 五、综合题（20分）
1. 内排序与外排序、简单选择排序
2. 快速排序的划分过程

## 答案特点

- 所有题目均提供详细解答
- 算法题包含完整的C语言代码实现
- 分析题包含详细的步骤说明
- 计算题包含完整的计算过程

## 使用说明

1. 先阅读 `练习卷1.pdf` 了解题目要求
2. 尝试独立完成练习题
3. 参考 `练习卷1-答案.md` 核对答案
